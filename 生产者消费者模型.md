#### 生成器函数 send方法传参给yield， yield语法接受参数的使用。
-   在生成器函数中定义的yield可以赋值给一个变量，而且yield可以接受参数。如果没有给yield传递参数，那么yield就为None，yield赋值的变量也为None。
```
import time
def chi(name):
    print('我是广东人[%s],我要开始吃福建人了' %name)
    while True:
        time.sleep(0.1)
        fujianren = yield
        print('广东人%s疯狂的把福建人%s吃了' %(name,fujianren))


def beichi():
    n1 = chi('ali')
    n2 = chi('lia')
    n1.__next__()
    n2.__next__()
    for i in range(10):
        time.sleep(0.1)
        n1.send('福建人%s' %i)
        n2.send('福建人%s' %i)

beichi()
```
